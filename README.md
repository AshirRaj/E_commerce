# 🛒 MegaStore - Complete E-Commerce Website

MegaStore is a fully responsive and interactive e-commerce website developed using **HTML5**, **CSS3**, and **Vanilla JavaScript**. It provides users with a modern shopping experience including product browsing, search, filtering, wishlist, shopping cart, and checkout functionality.

---

## 🚀 Features

### 🏠 Home Page
- Responsive navigation bar
- Hero image slider
- Featured categories
- Promotional banners

### 🛍️ Product Management
- 100+ Products
- Category-wise filtering
- Product search
- Sort products
- Grid/List view
- Product ratings
- Product badges (New / Sale)

### ❤️ Wishlist
- Add products to wishlist
- Remove wishlist items
- Wishlist counter

### 🛒 Shopping Cart
- Add to cart
- Remove from cart
- Increase/Decrease quantity
- Cart sidebar
- Automatic price calculation
- Shipping calculation
- Tax calculation
- Grand total calculation

### 💳 Checkout System
- Shipping Details Form
- Payment Information
- Order Review
- Multi-step checkout process

### 🔍 Search Functionality
- Real-time product search
- Category search
- Instant filtering

### 🎯 Product Filters
- Price Range
- Rating Filter
- Category Filter
- Sort by:
  - Name
  - Price Low → High
  - Price High → Low
  - Rating

### 📱 Responsive Design
- Mobile Friendly
- Tablet Friendly
- Desktop Friendly
